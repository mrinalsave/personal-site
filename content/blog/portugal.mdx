### writing a new post

drop an `.mdx` file in `content/blog/` with frontmatter like the block at the top of this file:

- **title** — shown in the post header and the index card
- **date** — `YYYY-MM-DD`, used for sorting (newest first)
- **category** — one primary type; becomes a filter
- **tags** — freeform list; each becomes a filter too
- **summary** — one line shown on the index card
- **cover** — optional `/public` image path for the card

### the body is just markdown

write paragraphs, **bold**, *italics*, and [links](/about/) the normal way. images reference files in `/public`:
